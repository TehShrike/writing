A brief look at my design process when building things for the web.

  1. Try to identify the problem that needs to be solved
  2. Do research
  3. Create some content
  4. Edit content
  5. Sketch ideas
  6. Write some words
  7. Edit words
  8. Sketch some more
  9. Build a prototype
  10. Test it with users
  11. Make it fast
  12. Make it readable
  13. Make it work for everyone you can in every context you can think of
  14. Ship it
  15. Test it with users
  16. Go back to step #1
